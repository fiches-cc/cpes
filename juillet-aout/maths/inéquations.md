# ↔️ Rappel: Inéquations

## Les Bases

### Polynomes

$$Δ = b^2 - 4ac$$

- Si Δ < 0 : Pas de solution à l'équation

  $$S = \emptyset$$

- Si Δ = 0 : Une seule solution

  $$S = \frac{-b}{2a}$$

- Si Δ > 0 : Deux solutions à l'équation

  $$S_1 = \frac{-b-\sqrt{Δ}}{2a}$$

  $$S_2 = \frac{-b+\sqrt{Δ}}{2a}$$
